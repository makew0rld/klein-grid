Klein Grid

The Klein Sexual Orientation Grid is a multidimensional assessment tool developed by Fritz Klein as an expansion of the Kinsey Scale. It measures sexual orientation across seven different dimensions, each evaluated for three time periods. Learn more on Wikipedia.

Scale Reference

For questions A-E (attraction, behaviour, fantasies, emotional preference, social preference):

1: other sex only 2: other sex mostly 3: other sex somewhat more 4: both sexes equally 5: same sex somewhat more 6: same sex mostly 7: same sex only

For questions F-G (lifestyle preference, self-identification):

1: heterosexual only 2: heterosexual mostly 3: heterosexual somewhat more 4: heterosexual/homosexual equally 5: homosexual somewhat more 6: homosexual mostly 7: homosexual only
